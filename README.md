# Stock Backend

Projeto de frontend para disciplina TAP

## Como instalar dependencias

```
npm install
```

## Como rodar o projeto

```
ng serve
```

## Como buildar o projeto

```
ng build
```

## Como construir e rodar a imagem

```
 docker build -t store_front . 
 docker run -d -p 4200:80 store_front
```
